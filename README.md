# HSK Deck Landing Page

Static website for HSK Deck, including the public product page, privacy policy, support information, and data safety disclosures.

## Pages

- `index.html` - app landing page
- `privacy-policy.html` - privacy policy
- `data-safety.html` - plain-language data safety summary
- `terms.html` - terms of use
- `support.html` - support and contact page
- `data-deletion.html` - local data deletion instructions

## Deploy

This site is static. Deploy the full folder to any static hosting provider.
